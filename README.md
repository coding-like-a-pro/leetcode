# leetcode

## How make PR
1. Clone this repo to your own repository
2. Create folder and name it by your github login 
3. Add your files code to created folder(2)
4. Make pull request

## PR requirements
1. Name must contain tasks name 
2. Link to tasks
3. Time/Space complexity
